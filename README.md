# 家长控制程序-windows版本

#### 介绍
家长控制程序,windows电脑版, 就最简单的方式 每隔30分钟休息30分钟, 或输入密码解锁. python + tk 跨平台

#### 功能
计时锁屏 -> 进入休息时间
休息时间倒计时结束后 -> 解锁屏幕 -> 开始新的一轮计时锁屏

锁屏后，可输入密码解锁，重新计时
可配置计时锁屏时间；休息时间；解锁密码
